# \# 🌸 Lily Wars

# 

# \*\*Версия:\*\* 0.9 (Июнь 2026)

# 

# Коллекционная карточная игра в стиле Gwent / Hearthstone с уникальными фракциями, конструктором колод и тактическим боем.

# 

# \## Текущее состояние (v0.9)

# 

# \### Реализовано:

# \- ✅ \*\*4 фракции\*\* с уникальными лидерами и \~25 картами в каждой (всего 101 карта)

# \- ✅ \*\*Конструктор колод\*\* с лимитами (25 карт / 170 влияния) и проверкой maxCopies

# \- ✅ \*\*Красивый выбор фракций\*\* с логотипами

# \- ✅ \*\*Полноценная одиночная игра против бота\*\*

# &#x20; - Доска из 2 линий × 4 слота на сторону

# &#x20; - Рука игрока + интерактивное размещение карт

# &#x20; - Эффекты при размещении (урон, исцеление, баффы)

# &#x20; - Расчёт силы линий и раундов

# &#x20; - Урон по здоровью лидера (30 HP)

# &#x20; - Простой, но играбельный ИИ бота

# &#x20; - Победа / Поражение с возможностью переигры

# \- ✅ Сохранение колод в localStorage

# \- ✅ Адаптивный интерфейс под Telegram Mini App

# 

# \### Планируется (будущие версии):

# \- Онлайн-мультиплеер (WebSocket)

# \- Анимации карт и эффектов

# \- Полноценная система эффектов и триггеров

# \- Турниры и рейтинги

# \- Мобильная оптимизация и новые фракции

# 

# \## Как играть (одиночная игра)

# 

# 1\. Создай колоду в конструкторе (или используй готовую)

# 2\. Выбери колоду → "Играть с ботом"

# 3\. В свой ход кликай по карте в руке → затем по линии (Левая / Правая)

# 4\. Карты размещаются с учётом ограничений линии

# 5\. Эффекты срабатывают автоматически при размещении

# 6\. Закончи ход → бот играет → раунд разрешается

# 7\. Побеждает тот, кто первым снизит здоровье противника до 0
