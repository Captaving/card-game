# Lily Wars

**Версия:** 0.9 (Июнь 2026)

Коллекционная карточная игра с механикой, вдохновлённой Gwent.

## Текущее состояние

### Реализовано

* ✅ 4 фракции с уникальными лидерами (\~25 карт в каждой, всего 101 карта)
* ✅ Конструктор колод (лимит 25 карт / 170 влияния + проверка максимального количества копий)
* ✅ Выбор фракций с логотипами
* ✅ Сохранение колод в localStorage
* ✅ Полноценное главное меню в тёмной теме
* ✅ Игровой экран в стиле Gwent (2 линии у каждого игрока)
* ✅ Базовая механика боя против бота (в разработке)

### В разработке (v0.9)

* Полная Gwent-подобная механика:

  * 3 раунда
  * Победа в раунде = +1 монетка (первый кто наберёт 2 — выигрывает)
  * Система Pass
  * Урон наносится картам на поле
  * Лидер с активной способностью (1 раз за игру)
  * Mulligan в начале игры
  * Добор карт после каждого раунда

## Как играть

1. Создай колоду в конструкторе
2. Выбери колоду → «Играть с ботом»
3. Размещай карты на левую или правую линию
4. Используй кнопку **ПАС**, когда не хочешь больше ходить
5. Побеждает тот, кто первым наберёт 2 победы в раундах
