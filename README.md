# 🌸 Lily Wars

Тактическая карточная игра с четырьмя уникальными фракциями: Рыцари Лилии, Прислужники Черной Лилии, Дриады Нидолиона и Орден Чародеев.

## Особенности
- Красивый редактор колод с карточным интерфейсом
- Выбор количества копий при добавлении карт
- Игра против бота с простым ИИ и разрешением боёв на флангах
- Подготовка к онлайн-режиму (WebSocket сервер включён)
- Полная база карт на основе оригинальной Excel-таблицы

## Локальный запуск (для теста)
```bash
cd client
# Откройте index.html в браузере (или используйте live server)
```

Для онлайн-игры:
```bash
cd server
npm install
node server.js
```

Затем откройте клиент и используйте онлайн-кнопку (пока в разработке, но сервер готов).

## Деплой на сервер
Смотрите подробный гайд в файле **DEPLOY.md**

## Как играть
1. Создайте колоду (максимум 25 карт, 170 влияния)
2. Сохраните её
3. Выберите колоду → Играть с ботом
4. Играйте карты на левый/правый фланг
5. Заканчивайте ход — бот отвечает, происходит подсчёт силы на флангах

Удачи в сражениях за Лилию! 🌸
